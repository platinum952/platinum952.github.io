# 素数判断
{: .message }

## 参考代码：
{% highlight js %}
int isprime(int num){
	int i, k;
	int isprime = 1;
	k = (int)sqrt((double)num);
	for (i = 2; i <= k; i++)
		if (num % i == 0)
		{
			isprime = 0;
			break;
		}
	return isprime;
}
{% endhighlight %}
